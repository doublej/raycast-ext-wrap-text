# Wrap Text

Raycast extension to wrap clipboard or selected text in XML-like tags.

## Features

- Wrap clipboard content in `<tag>...</tag>`
- Wrap selected text in place using Cmd+C / Cmd+V automation

## Commands

| Command | Arguments | Description |
|---------|-----------|-------------|
| Wrap Clipboard | tag name | Wrap clipboard content in XML-like tags |
| Wrap In Place | tag name | Wrap selected text in XML-like tags in place |

## Install

```bash
bun install
bun run build
bun run dev
```
